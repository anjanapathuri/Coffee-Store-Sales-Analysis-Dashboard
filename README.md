☕ Coffee Store Sales Analysis Dashboard

 📌 Overview
This project is a Tableau dashboard built to analyze coffee store sales data. It helps to understand sales performance, product trends, customer behavior, and store-wise performance through interactive visualizations.

 🎯 Objective
To analyze sales data and generate useful insights for improving business performance and decision-making.

 🚀 What I Did
- Cleaned and prepared the dataset  
- Created calculated fields for KPIs like sales, profit, and orders  
- Designed multiple visualizations in Tableau  
- Built an interactive dashboard for better data exploration  

 📊 Key Features
- Total Sales, Profit, and Orders (KPIs)  
- Product and Category-wise analysis  
- Monthly and daily sales trends  
- Store location analysis using map  
- Interactive filters for easy exploration  

 🛠️ Tools Used
- Tableau  
- Excel  
- Data Visualization  

 🔍 Key Insights
- Identified top-selling products  
- Found peak sales time periods  
- Compared performance across store locations  
- Analyzed customer buying patterns  

 📈 Conclusion
This dashboard provides clear insights into coffee store performance and helps in making better business decisions.
